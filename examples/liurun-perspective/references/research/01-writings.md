## 一、出版书籍

### 1. 《底层逻辑：看清这个世界的底牌》
- **出版时间**：2021年（第一部），2022年（第二部）
- **出版社**：机械工业出版社
- **核心论点**：
  - "底层逻辑 + 环境变量 = 方法论"——方法论会随环境变化而过时，但底层逻辑不变
  - 一个人心中应该有三种"对错观"：法学上的对错、经济学上的对错、个体利益最大化的对错
  - 事实有真假，观点无对错——区分事实、观点、立场和信仰
  - 人性的底层逻辑：人是自利的、趋利避害的、追求确定性的
  - 社会的底层逻辑：公平与效率的权衡
- **可信度**：一手（刘润原著）
- **备注**：该书成为2021-2022年最畅销的商业类书籍之一，多次加印

### 2. 《5分钟商学院》
- **出版时间**：2017年（系列丛书，分为《个人篇》《商业篇》《管理篇》《工具篇》）
- **出版社**：中信出版社
- **核心论点**：
  - 系统化整理商业知识，将复杂的商业概念浓缩为可快速消化的模块
  - "每个人都应该懂一点商业知识"——商业思维是现代人的基本素养
  - 涵盖模块：消费者行为、营销、定价、产品、组织管理、战略等
- **可信度**：一手（刘润原著）
- **备注**：源于其在得到App上的同名付费课程，累计订阅超50万

### 3. 《商业简史》
- **出版时间**：2020年
- **出版社**：中信出版社
- **核心论点**：
  - 商业的本质是"交易"，交易的本质是"信息不对称"
  - 商业进化的方向是不断降低"交易成本"
  - 从物物交换到平台经济，商业形态的每一次跃迁都与交易成本的降低有关
  - "商业的终极目标是降低一切不必要的交易成本"
- **可信度**：一手（刘润原著）

### 4. 《新零售：低价高效的数据赋能之路》
- **出版时间**：2017年
- **出版社**：中信出版社
- **核心论点**：
  - 新零售 = 更高的效率 + 更好的体验
  - "人、货、场"的重构——用数据打通零售的每一个环节
  - 零售的本质：信息流、资金流、物流的组合优化
  - 提出"坪效革命"概念——用线上数据赋能线下零售
- **可信度**：一手（刘润原著）
- **备注**：该书得到马云新零售概念的启发，刘润在此基础上构建了自己的理论框架

### 5. 《趋势红利：找到你的下一个风口》
- **出版时间**：2016年
- **出版社**：中信出版社
- **核心论点**：
  - 识别趋势比努力更重要——"在正确的方向上努力"
  - 四种红利：流量红利、社交红利、产品红利、组织红利
  - 趋势来临时要有"提前半步"的认知
- **可信度**：一手（刘润原著）

### 6. 《互联网+：传统企业的自我颠覆与价值重塑》
- **出版时间**：2015年
- **出版社**：中信出版社
- **核心论点**：
  - 互联网不是工具，而是基础设施
  - 传统企业转型的核心不是"加互联网"，而是用互联网思维重构商业模式
  - 提出企业转型的四个阶段：看不见、看不起、看不懂、来不及
- **可信度**：一手（刘润原著）

### 7. 《关键跨越》
- **出版时间**：2024年（约）
- **出版社**：待确认
- **核心论点**：
  - 企业成长中的关键转折点
  - 组织能力的系统性提升
- **可信度**：二手（部分细节待验证）

## 二、自创术语与概念

### 1. "底层逻辑"
- **定义**：事物背后不变的本质规律，是方法论之下的更深层结构
- **出处**：《底层逻辑》系列丛书
- **核心公式**：底层逻辑 + 环境变量 = 方法论
- **出现频率**：极高（贯穿其几乎所有后期著作和演讲）
- **可信度**：一手

### 2. "五商"（Five Quotients）
- **定义**：智商（IQ）、情商（EQ）、逆商（AQ）、美商（BQ）、财商（FQ）
- **出处**：《5分钟商学院》及多次演讲中提及
- **核心观点**：成功需要五种商数的综合发展，而非单一依赖智商
- **可信度**：一手

### 3. "商业杠杆"（Business Leverage）
- **定义**：用最少的资源撬动最大的商业价值
- **出处**：《底层逻辑》及年度演讲
- **核心观点**：杠杆的本质是找到"关键支点"——投入产出比最高的环节
- **可信度**：一手

### 4. "信息不对称"作为商业本质
- **定义**：商业的核心驱动力是信息差
- **出处**：《商业简史》
- **核心观点**：所有商业模式都可以归结为"消除信息不对称"或"利用信息不对称"
- **可信度**：一手

### 5. "交易成本"
- **定义**：完成一次交易所需的所有成本（搜寻成本、谈判成本、执行成本等）
- **出处**：《商业简史》，受科斯（Ronald Coase）交易成本理论启发
- **核心观点**：商业进化的方向是不断降低交易成本
- **可信度**：一手（概念源自科斯，刘润做了通俗化和应用化）

### 6. "人货场"重构
- **定义**：新零售的核心框架——重新定义消费者（人）、商品（货）、消费场景（场）
- **出处**：《新零售》
- **核心观点**：用数据打通人、货、场，实现零售效率的全面提升
- **可信度**：一手

### 7. "进化"作为核心隐喻
- **定义**：商业世界如同自然进化，适者生存
- **出处**：年度演讲系列主题
- **核心观点**：企业必须持续进化才能生存，进化不是选择而是必须
- **可信度**：一手

### 8. "未来已经到来，只是分布不均"
- **定义**：趋势已经在某些地方发生，只是还没有普及到所有地方
- **出处**：多次演讲和文章中引用（此句原为威廉·吉布森的名言，刘润做了商业化的阐释）
- **可信度**：一手（刘润的阐释），原始出处为威廉·吉布森

## 三、核心框架体系

### 刘润商业思维的整体框架

```
┌─────────────────────────────────────────┐
│           底层逻辑（不变的本质）           │
├─────────────────────────────────────────┤
│  人性底层  │  社会底层  │  商业底层       │
├─────────────────────────────────────────┤
│          方法论（随环境变化）              │
├─────────────────────────────────────────┤
│  战略  │  组织  │  产品  │  营销  │  管理  │
├─────────────────────────────────────────┤
│           工具（具体执行）                 │
└─────────────────────────────────────────┘
```

### 商业四要素框架
1. **交易**：商业的原子单位
2. **信息不对称**：商业的根本驱动力
3. **交易成本**：衡量商业效率的核心指标
4. **进化**：商业发展的永恒主题

### 组织能力框架
- 刘润强调组织能力是企业最核心的竞争力
- 组织能力 = 战略 × 组织 × 文化
- "战略决定天花板，组织决定地板"
